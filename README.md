### Installazione
Ho usato MiKTeX su MacBook con Vscode, estensione LateX Workshop. File unico [qui](logica.tex). Pdf generato [qui](logica.pdf). 

Tutorial [qui](https://www.youtube.com/watch?v=CmagZthwhaY). 

Con Miketex o anche altri:
```bash
pdflatex logica.tex 
```
